# 打磨规范（POLISH-SPEC）— 所有 Kit 统一执行

> 目标：去掉"AI 味"，达到生产级品质。综合 impeccable / frontend-design 的反 slop 标准。
> 红线：**不改 --ds-* token 契约、不破坏 AA 对比、不删已接入的 fonts.css、不删 7 类组件与移动端/响应式、不引任何新外链**。

## 规则 1 · 去 emoji，换内联 SVG（最重要）

- 把所有**彩色/象形 emoji**（🎀👾🏢📊🇨🇳🌌📖🫧✅❌🎉🚀⭐🔒📥🕵️🛡️📤🚦… 以及任何作图标/徽标/项目符号/状态/章节前缀/logo 的 emoji）**全部移除**，按用途替换为：
  - **内联 SVG 图标**：统一风格（同一 Kit 内一致），`stroke="currentColor"` 或 `fill="currentColor"`、`width/height:1em~24px`、线宽约 1.5–2、`viewBox="0 0 24 24"`；颜色继承上下文（用 currentColor，必要时 `var(--ds-*)`）。状态图标（成功/警告/危险/信息）用对应语义色的 SVG（对勾/感叹/叉/i）。
  - **章节标题前缀 emoji**：直接删除，只留文字标题（如 `🎨 色板` → `色板`）。
  - **logo/品牌位 emoji**：换成一个小的 SVG 品牌标记（可用该风格主色画个简洁几何/字母标），或纯文字 wordmark。
- **保留**的中性排版符号：箭头 `→ ← ↑ ↓`（用于范围/流向，属正常排版）可保留；但作"外链/跳转"的 `↗` 建议换 SVG。`✓ / ✗` 若用作表格状态，优先换 SVG 对勾/叉，至少要 monochrome 并用语义色。
- 完成后该文件 grep 不应再有彩色 emoji。

## 规则 2 · 修正 impeccable 明令禁止项

- **侧边色条 alert**：凡 alert/callout/list-item 用 `border-left`(或 right) ≥2px 的彩色竖条做强调 → 改为「**前置语义 SVG 图标 + 浅色底 `*-bg` + 1px 全描边**」。不要保留粗色条。
- **渐变文字**：`background-clip:text` + gradient 文字 → 改纯色（用 `--ds-fg` 或品牌主色），强调靠字重/字号。（形状/图标上的渐变填充可保留。）
- **ghost-card**：同一元素上「1px 描边 + box-shadow 模糊 ≥16px」并存 → 二选一（要么单一描边，要么一层 ≤8px 的阴影）。**例外见规则 3**。
- **display 字距下限**：标题 `letter-spacing` 不得低于 `-0.04em`（太紧会粘字）。
- **语义 z-index**：组件层级用语义尺度（dropdown<sticky<modal-backdrop<modal<toast<tooltip），**禁止 9999/999** 之类魔法值；可在 tokens 或 preview 内联定义 `--ds-z-*`。

## 规则 3 · 风格识别特征属"故意"，不要中和（关键）

下列是各风格的**身份**，**不是** slop，**禁止**因规则 2 把它们抹平——规则 2 只打真正的随手装饰：
- 像素风：0 圆角、**硬实心偏移阴影**(`Npx Npx 0 0`)、steps() ——保留。
- 新拟物：同底卡片、**双向柔和阴影/inset 凹陷**、大圆角 ——保留（这是其唯一表达手段）。
- 可爱风：大圆角药丸、**柔和彩色阴影** ——保留。
- 暗黑科技：**霓虹辉光阴影/glow focus**、暗色优先 ——保留（glow 不算 ghost-card）。
- 政府风：红色 banner/标题栏、方正小圆角 ——保留。
- 你的 Kit 具体豁免见任务里那一行。

## 规则 4 · craft 打磨

- 间距讲节奏（组内/组间/区块层级分明），对齐统一，章节标题样式一致。
- 正文行宽 ≤ 75ch；占位文字也要 ≥4.5:1（别用浅灰糊住）。
- 卡片别滥用、别嵌套卡片。
- 图标对齐基线，尺寸统一；hover/选中/激活/禁用态都要在。
- 保持各风格气质鲜明（一眼可辨），不要打磨成千篇一律。

## 规则 5 · 自检（必须做）

1. `grep` 确认该 Kit 的 preview.html 与 README.md **无彩色/象形 emoji 残留**。
2. HTML 标签闭合配平；除 `../../_fonts/fonts.css` 外**无外链**。
3. 亮/暗（及 aibacknovel 三主题、dark-tech/agent 暗色优先）下文字与图标对比 **≥4.5:1**（大字/图标 ≥3:1）。
4. fonts.css 链接、7 组件、移动端块、响应式断点**仍在**；--ds-* 契约未改名。

回报：1–2 句说明替换了多少 emoji、改了哪些禁止项、用 SVG 图标统一了什么。不解释过程。
