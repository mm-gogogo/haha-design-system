The Candle Index: Notes on Paper, Oxblood, and the Scholarly Interface

Iris Morrow · 2026-06-18 · catalogue MV-34-019

A dark-academia interface should read like a working archive: refined, dense, and legible, with ornament reserved for moments of authentication.

When a reader opens the Morrow & Vale catalogue, the first signal is paper rather than chrome.1 The surface is aged, the ink is warm, and the action color is oxblood because it feels stamped, not clicked. Bottle green appears in links and notes, where a curator would naturally point from text to shelf.

The system is intentionally more compact than an airy reading-blog style. It allows ruled lines, catalogue numbers, small-caps headings, and side notes, but each motif has a job. The interface should feel handled by careful archivists, never decorated for its own sake.

Footnotes Are Navigation, Not Decoration

Superscript references use the mono stack so they read as catalogue mechanics.2 The footnote list receives a strong top rule, turning the bottom of the text into a small archive ledger. Marginalia sits beside the reading column on desktop and falls below it on mobile, preserving a 390px viewport without horizontal drift.

"A margin is not empty. It is the room where the reader leaves a second voice."

Invented note from the Morrow & Vale house proof

Brass Remains Ceremonial

Brass is reserved for the seal, the fleuron, the caption rule, and the dark-theme strong border. It is not used for running text. Body copy stays on paper and ink; large decorative marks can carry brass because their purpose is ceremonial rather than informational.3

/* Scholarly reading contract */
.article-layout {
  grid-template-columns: minmax(0, 680px) minmax(180px, 240px);
}
.article sup.ref {
  font-family: var(--ds-font-mono);
  color: var(--ds-primary);
}
  1. Manuscript surfaces use `--ds-bg`, `--ds-bg-subtle`, and `--ds-bg-elevated`; no component needs hardcoded paper colors.
  2. Footnote refs are intentionally small but high contrast, using oxblood rather than brass.
  3. Decorative brass is `--ds-brass`; keep it on seals, rules, and captions rather than body copy.

Iris Morrow · 2026-06-18 · MV-34-019

我们常误以为「设计」意味着添加——更多色彩、更多阴影、更多动效。但在阅读的语境里，真正困难的是克制：知道哪些东西不该出现。

排版优先意味着，正文不是装饰，而是产品本身。于是字号要够大、行高要够松、每行的字数要控制在三十多个汉字之内——超过这个宽度，眼睛回到行首时就容易迷路。
