快速上手

哈哈图压把图片压缩全部放在本地完成，不上传、不打水印。安装后把图片拖进窗口即可，或用命令行批量处理。

pnpm add @haha/core @haha/imgsqueeze
haha squeeze ./photos --quality 78 --out ./dist

所有处理默认本地完成；若启用上传类功能，界面会显式告知并可一键关闭。

目标与北极星

哈哈的使命是把「小而美」的好工具做成免费且开源的全家桶。北极星指标是月活跃创作者数与赞助续费率，而非下载量虚荣指标。我们克隆 少数派 / 数码荔枝 / Keka 等里真正有用的工具，去掉付费墙与广告。

非目标

不做大而全的平台霸权，不收集用户数据，不把网盘推广伪装成主下载。所有「赞助 · 推广」位均统一中性标注。

技术约束

统一底座 Tauri + Next.js+WASM，媒体走 ffmpeg / yt-dlp / aria2。合规：全开源下 AGPL 条件可用，GPL 走子进程。

pnpm add @haha/core @haha/ui @haha/media
# 启动开发
pnpm dev --filter @haha/imgsqueeze

I3 · 安全需求

所有处理默认本地完成；上传类功能须显式告知并可关闭。
